# Assets Directory

Place your images and static files here.

## Required Images

1. **Profile Image**: `profile.jpg` (recommended: 400x400px, square)
   - Used in the Hero section

2. **Project Images**: `project1.jpg` through `project6.jpg` (recommended: 800x600px)
   - Used in the Projects section
   - You can use fewer images if you have fewer projects

## Image Recommendations

- **Format**: JPG or PNG
- **Profile Image**: Square format (1:1 aspect ratio)
- **Project Images**: Landscape format (4:3 or 16:9 aspect ratio)
- **Optimization**: Compress images for web to improve loading times

## Placeholder Images

The portfolio is configured to use placeholder images from `via.placeholder.com` if the actual images are not found. You can replace these with your own images by:

1. Adding your images to this directory
2. Updating the image paths in `src/data/portfolio.js`
